## 概述

pillow_crawler是一个轻量、易用、易扩展的爬虫框架，可以处理日常的简单爬虫任务。pillow_crawler非常轻量，只需要有python环境即可运行。同时pillow_crawler也支持扩展成分布式集群模式。


## 系统要求

Python 3.4+

依赖库：
* yaml：``` pip install PyYaml```
* mysqlclient: ```pip install mysqlclient```

## 运行

```
cd samples
python tmall_anta_crawler.py
```
